# 🎤 Voice Assistant using Python
A simple and efficient **voice-controlled assistant** built using Python that can perform tasks like opening applications, browsing websites, and responding to user commands using speech.

## 🚀 Features
* 🎙️ Speech Recognition (voice input)
* 🔊 Text-to-Speech response
* 🌐 Open websites:
  * YouTube
  * Google
  * Gmail
  * ChatGPT
  * GitHub
* 💻 Open applications:
  * Notepad
  * Calculator
  * Paint
  * Command Prompt

## 🛠️ Technologies Used
* Python
* SpeechRecognition
* pyttsx3
* webbrowser
* subprocess

## ▶️ How to Run
1. Install required libraries:
pip install SpeechRecognition pyttsx3 pyaudio
2. Run the program:
python voice-command.py

## 📌 Example Voice Commands
* open youtube

* open github

* what is the time
